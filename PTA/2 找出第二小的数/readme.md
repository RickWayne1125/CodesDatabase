# 找出第二小的数

求n个整数中第二小的数。
相同的整数看成一个数。比如，有5个数分别是1,1,3,4,5，那么第二小的数就是3。
输入格式:

输入包含多组测试数据。输入的第一行是一个整数C，表示有C组测试数据；
每组测试数据的第一行是一个整数n，表示本组测试数据有n个整数（2<=n<=10），接着一行是n个整数（每个数均小于100）。
输出格式:

为每组测试数据输出第二小的整数，如果不存在第二小的整数则输出“NO”，每组输出占一行。
## 输入样例:

在这里给出一组输入。例如：
```
3
2
1 2
5
1 1 3 4 5
3
1 1 1
```
## 输出样例:

在这里给出相应的输出。例如：
```
2
3
NO

```
